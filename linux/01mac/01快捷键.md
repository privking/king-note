# Mac快捷键

## 打开连接服务器

```
cmd+k
```

## 打开新的访达

```
cmd+n
```

## 删除选中文件

```
cmd+backspace
```

## 退出

```
cmd+q
```

## finder 查看/关闭隐藏文件

```
cmd+shift+.
```

## Mac安装软件提示`已损坏无法打开`,`请移到废纸篓`

```
sudo spctl --master-disable
```

以上操作是为了打开：系统偏好设置 – 安全性与隐私 – 通用 ，中的“任何来源”选项。
